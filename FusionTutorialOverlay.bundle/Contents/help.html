AI Tutorial Overlay for Fusion 360

Overview

This add-in provides AI-generated step-by-step tutorials directly inside Fusion 360, helping beginners learn CAD through guided exercises.

Features

  • Step-by-step instruction cards with rationale
  • Animated cursor demonstrations
  • 3D viewport interaction (camera, selection prompts)
  • Quality check symbols for self-assessment

Getting Started

  1. Open Fusion 360
  2. Click the "AI Tutorial Overlay" button in the toolbar
  3. The tutorial panel will open on the right side
  4. Follow the step-by-step instructions

Navigation

  • Next/Previous: Move between steps
  • Replay: Replay the cursor animation for the current step
  • Step dots: Jump to any step directly

Symbol Legend

  • ✅ Good / Correct
  • ⚠️ Caution / Check this
  • ⛔ Stop / Fix before continuing
